## 3. 기능 명세 🗂
<details>
<summary> 주요 기능 🖋 </summary>
<div markdown="1">       

(1) 장소 검색 

- 장소 검색 기능 제공
- 사용자 위치 기반 검색 기능 제공

(2) 카테고리 별 장소 조회

- 검색어 지정 검색 (가게 이름 일치, 가게 이름 포함, 도로명 주소, 업종, 리뷰n개 이상/이하, 평점n개 이상,이하)
- 랭킹 조회 (리뷰 적은순, 리뷰 많은 순, 평점 낮은 순, 평점 높은 순)

(3) 예약(웨이팅) 시스템

- 음식점/카페 웨이팅 신청 기능 제공
- 웨이팅 신청 시 현재 현황(잔여 자리) 및 현재 현황(대기 순서) 확인 가능

(4) 장소 리뷰 기능

- 장소 리뷰 작성, 리뷰 조회시 리뷰 갯수 확인 가능

</div>
</details>
